# 유저 스토리

| 기능 | 스토리 | 스팩 |
| --- | --- | --- |
| 온보딩 |  |  |
|  | 유저는 <br>앱의 기능을 파악하기 위해, <br>온보딩 이미지를 확인할 수 있다 | 온보딩 이미지를 모두 확인하였을 때, 로그인이 가능 |
| 로그인 |  | 유저가 존재하면 <br>- 로그인<br>존재하지 않으면<br>- 회원가입 |
|  | 유저는 <br>카카오톡을 사용해서 로그인을 할 수 있다. | 카카오톡이 설치되어있을때<br>- 카카오톡 앱을 활용하여 인증<br>설치되어있지 않을때<br>- 스토어로 이동 |
|  | 유저는 <br>애플 계정을 사용해서 로그인을 할 수 있다. |  |
| 닉네임 설정 |  |  |
|  | 유저는 <br>상대방에게 보일 이름을 지정하기 위해, <br>닉네임을 설정할 수 있다. | 닉네임 정책<br>- 20자<br>- 영어, 특수문자, 이모지 가능 |
| 매칭 |  |  |
|  | 초대 받은 유저는 <br>초대 링크의 상대방 ID로 매칭을 할 수 있다. |  |
| 초대장 보내기 |  |  |
|  | 유저는 <br>투투 메이트와 매칭을 하기 위해, <br>초대장을 보낼 수 있다. | 공유 방식<br>- 시스템 내장 공유 기능<br><br>링크 포맷 정책<br>- https://domain/{내본인 ID}<br>- 사용기술 : 딥퍼드 딥링크<br>   - Firebase Dynamic Link |
| 설명서 |  |  |
|  | 유저는 <br>서비스의 기능 및 의도를 파악하기 위해, <br>설명서를 확인할 수 있다. |  |
| 챌린지 확인 |  |  |
|  | 유저는 <br>현재 진행중인 챌린지의 진행도를 파악할 수 있다. | 진행도 확인 방식<br>- 프로그래스 (퍼센테이지)<br>- 새싹 사이즈<br><br>진행도 정책<br>- 5번 : 새싹 성장<br>- 10번 : 새싹 성장<br>- 15번 : 새싹 성장<br>- 17번 : 꽃<br>- 22번 : 활짝핀 꽃<br><br>퍼센테이지 정책<br>  - 1번 : 5%<br>  - 19번 : 95%<br>  - 20번 : 99%<br>  - 21번 : 99%<br>  - 22번 : 100% |
| 찌르기 |  |  |
|  | 유저는 <br>투투 메이트에게 챌린지를 독촉하기 위해, <br>찌르기를 할 수 있다. | 찌르기 횟수<br>- 일일 5회<br><br>찌르기 문구 정책<br>- 30자<br>- 영어, 특수문자, 이모지 가능<br><br>푸시 정책<br>- 찌르기 문구를 푸시 메세지에 표출 |
| 챌린지 생성 |  | 챌린지 인증 안할시 오후 9시 노티 |
|  | 유저는 <br>챌린지의 기본적인 정보를 작성할 수 있다. | 정보 요소<br>- 챌린지 이름<br>- 시작일<br>- 종료일<br><br>챌린지 이름 정책<br>- 20자<br>- 영어, 특수문자, 이모지 가능<br><br>시작일, 종료일 선택 정책<br>- 시작일 선택시 +22일로 종료일 설정<br>- 종료일 선택시 -22일로 시작일 설정 |
|  | 유저는 <br>룰, 벌칙 등 구체적인 챌린지 정보를 추가하기 위해, <br>추가적인 챌린지의 정보를 작성할 수 있다. | 추가 정보는 선택적으로 입력 가능 |
|  | 유저는 <br>어떤 챌린지를 생성할 지 추천 받기 위해, <br>서비스가 추천하는 챌린지를 확인 및 선택할 수 있다. |  |
| 챌린지 정보 확인 |  |  |
|  | 유저는 <br>작성된 챌린지 기본 정보 및 추가정보를 확인할 수 있다. | 정보<br>- D-n<br>- 챌린지명<br>- 챌린지 추가정보 |
| 꽃 선택 |  |  |
|  | 유저는 <br>상대방이 키울 꽃을 선택하기 위해, <br>꽃들을 확인할 수 있다. | 꽃 정보<br>- 사진<br>- 이름<br>- 꽃말 |
| 챌린지 그만두기 |  |  |
|  | 유저는 <br>현재 진행중인 챌린지를 그만둘 수 있다. |  |
|  | 유저는 <br>상대방이 키울 꽃을 선택할 수 있다. | 생성 시점이 겹쳤을 때는 이후에 생성한 유저에게 피드백<br>- 메인 화면으로 이동, 토스트 메세지 |
| 인증 |  |  |
|  | 유저는 <br>오늘 챌린지를 인증하고 히스토리를 남기기 위해, <br>인증을 할 수 있다 | 사진<br>- 크롭 기능<br>- 필수 1장<br>- 편집 가능<br>- 갤러리, 카메라 선택 가능<br><br>소감<br>- 최대 100자<br>- 필수 1자<br><br>인증시 노티 보내기<br>- OO님이 인증을 완료하였습니다. 확인해보세요 |
| 칭찬 |  |  |
|  | 유저는 <br>오늘 인증을 서로 완료했음을 축하하기 위해, <br>칭찬을 남길 수 있다. | 최대 20자<br><br>칭찬시 노티 보내기<br>- OO님이 칭찬을 남기셨어요. 확인해보세요 |
| 히스토리 |  |  |
|  | 유저는 <br>과거 챌린지 리스트를 조회할 수 있다. | 챌린지 이름 표시<br>서로가 키운 새싹/꽃 심어짐 |
|  | 유저는 <br>해당 챌린지의 인증 기록을 확인할 수 있다. |  |
|  | 유저는 <br>인증 상세에 대한 정보를 확인할 수 있다. | 정보<br>- 날짜<br>- 사진<br>- 챌린지명<br>- 소감<br>- 칭찬문구 |
| 내 정보 조회 |  |  |
|  | 유저는 <br>내 정보를 확인할 수 있다. | 정보<br>- 이름<br>- 개인정보처리취급방침<br>- 개발자 정보<br>- 문의하기<br>- 로그아웃 |
